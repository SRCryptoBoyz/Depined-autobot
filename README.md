# Depined-autobot

## Installation

1. Clone this repository

```bash
git clone https://github.com/SRCryptoBoyz/Depined-autobot.git
```

```
cd Depined-autobot
```
2. Install dependencies

```bash
npm install
```
3. Fill Data and Proxy (Optional)

```bash
nano data.txt
```

```
nano proxy.txt
```
4. Run Bot

```bash
npm run start
```
